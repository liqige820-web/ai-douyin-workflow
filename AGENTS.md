# AGENTS.md

本项目用于搭建“每日自动生成抖音 AI 短视频草稿”的零代码工作流。任何后续协作者、自动化代理或人工维护者，都必须遵守本文件。

## 总目标

每天固定时间自动完成：

1. 联网搜索 AI 热点。
2. 生成抖音短视频选题、脚本、标题、封面文案。
3. 从 Pexels、Pixabay 等授权素材库查找素材。
4. 使用 OpenAI 生成配音。
5. 使用 Creatomate 合成 1080x1920 竖版视频草稿。
6. 使用 n8n 定时执行。
7. 使用 Google Sheets 记录全过程。
8. 生成发布前人工审核清单。

## 硬性规则

- 不自动发布到抖音，只生成草稿和审核清单。
- 不搬运、不洗稿、不下载未授权视频。
- 所有事实性内容必须保留来源链接。
- 标题、字幕、口播、封面文案、视频画面里都不能出现微信号、手机号、QQ群、QQ号、Telegram、WhatsApp、邮箱等站外联系方式。
- 不生成虚假事实、夸大承诺、医疗/金融/法律等高风险建议。
- 不使用未经授权的人脸、明星肖像、商标素材作为主要视觉。
- 不把 API Key、账号密码、密钥写进文档、表格公开列、视频字幕或脚本。

## 内容标准

每条视频草稿必须包含：

- 选题一句话。
- 目标观众。
- 事实来源链接，至少 2 条，越关键的事实越要有来源。
- 口播脚本。
- 字幕文案。
- 标题候选。
- 封面文案。
- 素材来源链接和授权平台。
- AI 配音文件链接或路径。
- Creatomate 渲染结果链接或路径。
- 人工审核结论。

## 审核优先级

如果自动化结果出现以下情况，状态必须改成 `需要人工修改`：

- 来源链接缺失。
- 来源内容和脚本事实不一致。
- 出现站外联系方式。
- 素材来源不是授权素材库。
- 标题过度夸张或像谣言。
- 事实性内容无法核验。
- 视频渲染失败或字幕遮挡严重。

## 文件使用约定

- `docs/`：操作说明和设计文档。
- `n8n/`：以后放 n8n 工作流 JSON 导出文件。
- `sheets/`：Google Sheets 表头和字段说明。
- `prompts/`：以后放给 OpenAI 的提示词模板。
- `assets/`：素材来源记录，不建议长期保存大文件。
- `outputs/`：本地测试生成的音频、视频、封面。
- `review/`：人工审核清单模板和审核记录。
- `secrets/`：仅用于本地说明，不提交真实密钥。
