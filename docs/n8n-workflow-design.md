# n8n 工作流设计说明

工作流名称：`每日 AI 抖音短视频草稿`

目标：每天固定时间自动生成短视频草稿，但不自动发布抖音。

## 推荐流程总览

```text
定时触发
→ 搜索 AI 热点
→ 清洗和去重热点
→ 生成选题
→ 生成脚本、标题、封面文案
→ 检查事实来源
→ 检查站外联系方式
→ 搜索授权素材
→ 生成 OpenAI 配音
→ 提交 Creatomate 合成视频
→ 等待渲染完成
→ 写入 Google Sheets
→ 生成人工审核清单
→ 停止，等待人工发布
```

## n8n 节点设计

### 1. Schedule Trigger

用途：每天固定时间启动。

建议设置：

- Trigger Interval：`Days`
- Hour：你想每天生成草稿的小时，例如 `09`
- Minute：例如 `00`
- Timezone：`Asia/Shanghai`

### 2. Set：基础配置

用途：统一保存当天任务参数。

建议字段：

- `topic_count`：每天生成几个选题，先填 `3`
- `video_count`：每天合成几个视频，先填 `1`
- `language`：`zh-CN`
- `video_size`：`1080x1920`
- `platform`：`douyin`
- `must_keep_sources`：`true`
- `manual_review_required`：`true`

### 3. HTTP Request：搜索 AI 热点

用途：联网获取 AI 新闻、产品更新、研究进展、行业热点。

零代码阶段建议先用这些来源：

- 官方博客 RSS 或网页
- 可信科技媒体 RSS
- Google News 搜索结果
- 产品官方更新页面

输出必须保留：

- 热点标题
- 摘要
- 来源名称
- 来源链接
- 发布时间

### 4. Code 或 Item Lists：清洗和去重

用途：去掉重复热点、过旧内容和来源不明内容。

保留规则：

- 必须有链接。
- 优先保留 24-72 小时内的新内容。
- 同一事件多来源时保留权威来源。
- 低可信来源只作参考，不直接写成事实。

### 5. OpenAI：生成选题

用途：把热点变成适合抖音短视频的选题。

输出字段：

- `topic`
- `angle`
- `target_audience`
- `why_now`
- `risk_note`
- `source_links`

### 6. Google Sheets：写入选题池

用途：把选题先记录下来，方便人工追踪。

目标分页：`选题池`

写入状态：

- `待生成脚本`

### 7. OpenAI：生成脚本、标题、封面文案

用途：生成 30-60 秒竖版短视频内容。

要求：

- 口播要自然。
- 每个事实都能追溯到来源链接。
- 标题不要夸大。
- 字幕不要出现站外联系方式。
- 结尾不要引导加微信、加群、私信领资料。

### 8. IF：站外联系方式检查

用途：如果出现违规联系方式，直接打回。

必须拦截的内容：

- 微信、VX、V 信、wechat
- 手机号
- QQ、QQ群
- Telegram、TG
- WhatsApp
- 邮箱
- “加我”“进群”“私信领资料”等强导流话术

失败状态：

- `需要人工修改`

### 9. IF：事实来源检查

用途：确保事实性内容有来源链接。

通过条件：

- 至少 2 条来源链接。
- 关键数据、发布时间、模型名称、公司名称有对应来源。

失败状态：

- `来源不足`

### 10. HTTP Request：搜索 Pexels / Pixabay 素材

用途：根据脚本关键词搜索授权图片或视频。

必须记录：

- 素材平台
- 素材 ID
- 素材页面链接
- 下载链接
- 作者名
- 授权说明链接

注意：不要用搜索引擎随便下载视频。

### 11. OpenAI：生成配音

用途：把最终口播脚本转成音频。

建议：

- 模型：`gpt-4o-mini-tts`
- 声音：先用 `coral` 或你之后选定的声音
- 输出：`mp3`
- 语气：清晰、可信、节奏偏短视频

配音文字中应明确避免：

- 站外联系方式
- 未核实结论
- 绝对化承诺

### 12. HTTP Request：提交 Creatomate 渲染

用途：把素材、字幕、配音、标题交给 Creatomate 合成视频。

视频规格：

- 宽：`1080`
- 高：`1920`
- 比例：`9:16`
- 格式：`mp4`

### 13. Wait + HTTP Request：等待渲染完成

用途：Creatomate 渲染需要等待，不能马上拿到视频。

建议：

- 每 30 秒查询一次。
- 最多等待 10 分钟。
- 超时就写入 `渲染超时`。

### 14. Google Sheets：更新视频草稿

用途：写入最终草稿信息。

目标分页：`视频草稿`

状态建议：

- `待人工审核`
- `需要人工修改`
- `渲染失败`

### 15. Google Sheets：生成人工审核清单

用途：提醒你发布前逐项检查。

目标分页：`人工审核`

必须默认写入：

- `未审核`

## 异常处理

如果某一步失败，不要继续发布流程，只更新 Google Sheets 状态：

- 搜索失败：`热点搜索失败`
- 来源不足：`来源不足`
- 素材失败：`素材未找到`
- 配音失败：`配音失败`
- 渲染失败：`渲染失败`
- 合规失败：`需要人工修改`
