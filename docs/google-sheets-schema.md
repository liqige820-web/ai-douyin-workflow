# Google Sheets 表结构

请新建一个 Google Sheet，文件名建议：

`每日 AI 抖音短视频草稿库`

然后在底部创建 6 个分页：

1. `选题池`
2. `视频草稿`
3. `素材库`
4. `事实来源`
5. `人工审核`
6. `配置`

## 分页 1：选题池

第一行表头：

```text
记录ID,日期,热点标题,热点摘要,来源名称,来源链接,发布时间,选题,切入角度,目标观众,为什么现在发,风险提示,状态,脚本ID,备注
```

状态建议：

- `待生成脚本`
- `已生成脚本`
- `放弃`
- `需要人工判断`

## 分页 2：视频草稿

第一行表头：

```text
脚本ID,日期,选题,口播脚本,字幕文案,标题候选1,标题候选2,标题候选3,封面文案,画面分镜,配音文本,OpenAI配音状态,音频链接或路径,Creatomate模板ID,Creatomate渲染ID,视频链接或路径,视频状态,审核状态,发布状态,发布时间,抖音作品链接,备注
```

视频状态建议：

- `待配音`
- `配音完成`
- `待渲染`
- `渲染中`
- `待人工审核`
- `需要人工修改`
- `已废弃`

发布状态建议：

- `未发布`
- `人工已发布`
- `暂不发布`

## 分页 3：素材库

第一行表头：

```text
素材ID,脚本ID,日期,平台,素材类型,搜索关键词,素材标题,作者,素材页面链接,下载链接,授权说明链接,是否已核验授权,使用位置,备注
```

平台建议：

- `Pexels`
- `Pixabay`

素材类型建议：

- `video`
- `image`

## 分页 4：事实来源

第一行表头：

```text
来源ID,脚本ID,日期,事实点,来源标题,来源机构,来源链接,发布时间,引用位置,是否已人工核验,备注
```

引用位置示例：

- `开头`
- `第2句`
- `字幕第3屏`
- `标题`

## 分页 5：人工审核

第一行表头：

```text
审核ID,脚本ID,日期,视频链接或路径,事实是否有来源,来源是否可信,素材是否授权,是否无站外联系方式,标题是否不过度夸张,字幕是否无错别字,配音是否自然,画面是否无侵权风险,是否适合发布,审核人,审核结论,修改意见,审核时间
```

审核结论建议：

- `通过`
- `修改后再审`
- `不发布`

## 分页 6：配置

第一行表头：

```text
配置项,配置值,说明,是否启用,更新时间
```

建议先填这些配置：

```text
daily_topic_count,3,每天生成几个候选选题,是,
daily_video_count,1,每天合成几个视频草稿,是,
timezone,Asia/Shanghai,定时任务时区,是,
video_width,1080,视频宽度,是,
video_height,1920,视频高度,是,
manual_review_required,true,是否必须人工审核,是,
auto_publish,false,是否自动发布抖音，必须保持 false,是,
```

## 零代码复制方法

1. 打开 Google Sheets。
2. 新建空白表格。
3. 把第一个分页改名为 `选题池`。
4. 复制本文件对应分页的“第一行表头”。
5. 点 A1 单元格。
6. 粘贴。
7. 重复创建其他分页。
